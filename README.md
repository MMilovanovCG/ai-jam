# Claude Agent — Digital Helper Slides

Single-page slide deck describing an internal Slack bot that auto-reviews PRs, creates Jira tickets from DMs and threads, and answers questions in-channel.

## View

Open `index.html` in a browser, or serve it locally:

```bash
python3 -m http.server 8765
# then open http://localhost:8765/
```

## Controls

- **Arrow keys / PageUp / PageDown** — navigate slides
- **Ctrl+F** — toggle fullscreen presentation mode
- **Home / End** — jump to first / last slide
